psychology ออกแบบคำสั่ง AI (Kaizen Prompt Generator)

เลือกแม่แบบสำเร็จรูปสำหรับการเพิ่มผลผลิต ลดความสูญเสีย และแก้ไขปัญหาหน้างานด้วยพลังของ Gen AI

1. เลือกแม่แบบการเขียน Prompt

2. ปรับแต่งรายละเอียดปัญหาของคุณ

3. ผลลัพธ์ชุดคำสั่ง AI Prompt

โหมดจำลอง (Simulated Mode)
smart_toy ปรับแต่งคำสั่ง Prompt ทางด้านซ้าย จากนั้นกดปุ่ม "รันคำสั่งด้วย AI" ด้านล่างเพื่อดูแนวทางการวิเคราะห์ Kaizen ได้ทันที!

*การรันในโหมดจำลองไม่มีค่าบริการและรวดเร็ว สำหรับผลการวิเคราะห์ระดับลึกและเรียลไทม์ เชื่อมต่อ API Key ของคุณ

keyตั้งค่า Google Gemini API Key

คุณสามารถนำ Google Gemini API Key มาต่อเชื่อมเพื่อเรียกใช้บอทประมวลผลคำสั่งตรงถึง Google ได้ฟรี! โดย API Key จะถูกเก็บบันทึกไว้อย่างปลอดภัยเฉพาะในบราวเซอร์เครื่องนี้ของคุณเท่านั้น (LocalStorage)

วิธีรับ API Key ฟรี:

  1. ไปที่ Google AI Studio
  2. ล็อกอินด้วยบัญชี Google ทั่วไป
  3. กดปุ่ม "Get API Key"
  4. คลิก "Create API Key" และคัดลอกมาวางในหน้าต่างนี้ได้เลย!
