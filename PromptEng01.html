Kaizen × Prompt Engineering

เพื่อให้ได้ผลลัพธ์จาก AI แม่นยำ และนำไปใช้งานได้จริง

🎴 เทคนิคหลัก Kaizen → Prompt

ลด: ลดคำอธิบายที่ไม่จำเป็น ลดการทำซ้ำ
เปลี่ยน: เปลี่ยนจากข้อความล้วน → ตาราง/รายการ

📊
Big Q: QCDSMPEE
กำหนดมิติคำตอบ
เลือกมิติที่ต้องการให้ AI ตอบ เช่น คุณภาพ ต้นทุน เวลา ความปลอดภัย
Template Prompt
วิเคราะห์ [หัวข้อ] โดยพิจารณาตามกรอบ Big Q:
Q - คุณภาพ: ผลกระทบต่อมาตรฐาน
C - ต้นทุน: ประหยัดหรือเพิ่มค่าใช้จ่าย
D - เวลา: ช่วยลดหรือเพิ่มเวลาไหม
S - ความปลอดภัย: มีความเสี่ยงอะไร
ตอบเป็นตารางเปรียบเทียบ ก่อน/หลัง
💡 ทำไมได้ผล: กรอบ QCDSMPEE บังคับให้ AI คิดรอบด้าน ไม่มองแค่มุมเดียว
▼
📋
6 ขั้นตอน Kaizen
Step-by-Step
ให้ AI ทำงานตามขั้นตอน ค้นหา→สำรวจ→คิด→เลือก→ลงมือ→แสดงผล
Template Prompt
ช่วยปรับปรุง [งาน/กระบวนการ] โดยทำตาม 6 ขั้น:
1. ค้นหา: ระบุปัญหาที่พบ
2. สำรวจ: สภาพปัจจุบันเป็นอย่างไร
3. คิด: แนวทางแก้ไข 3 วิธี
4. เลือก: วิธีที่ดีที่สุด พร้อมเหตุผล
5. ลงมือ: แผนปฏิบัติจริง
6. ผลลัพธ์: ผลที่คาดว่าจะได้รับ
💡 ทำไมได้ผล: AI ไม่ข้ามขั้นตอน คำตอบมีโครงสร้างชัดเจน นำไปใช้ได้ทันที
▼
✍️
ปัญหา→แก้→ผลลัพธ์
โครงสร้างข้อเสนอแนะ
บังคับให้ AI ตอบครบ 3 ส่วนเหมือนแบบฟอร์ม Kaizen จริง
Template Prompt
ตอบในรูปแบบข้อเสนอแนะ Kaizen ดังนี้:

🔴 ปัญหา (ก่อน):
อธิบายสภาพปัญหา ระบุตัวเลข/เวลาที่สูญเสีย

🟢 วิธีแก้ (รายละเอียด):
อธิบายแนวทางแก้ไขอย่างเป็นรูปธรรม ทำได้จริง

✅ ผลลัพธ์ (หลัง):
ระบุประโยชน์ทั้งตัวเลข และคุณภาพชีวิต

หัวข้อ: [ระบุงานที่ต้องการปรับปรุง]
💡 ทำไมได้ผล: โครงสร้างนี้บังคับ AI คิดครบทุกมุม ไม่ตอบกว้างเกินไป

🧠
Limiting Beliefs → Positive
เปลี่ยนมุมมอง
ใช้หลักเปลี่ยนความเชื่อด้านลบ เพื่อให้ AI คิดเชิงสร้างสรรค์
Template Prompt
ฉันมีความเชื่อด้านลบว่า "[ความเชื่อที่ขัดขวาง]"

ช่วยทำ 3 อย่างนี้:
1. อธิบายว่าทำไมความเชื่อนี้ไม่เป็นความจริง
2. เปลี่ยนเป็นความเชื่อด้านบวกที่สร้างสรรค์
3. ยกตัวอย่างคนที่ทำสำเร็จแม้มีข้อจำกัดแบบนี้

บริบท: [สถานการณ์ของคุณ]
💡 ทำไมได้ผล: AI จะไม่ตอบแค่ "ทำได้" แต่จะให้เหตุผลและวิธีเอาชนะจริงๆ
▼
👨‍💼
สวมบทบาท Expert
Role + Experience
กำหนดให้ AI รับบทผู้เชี่ยวชาญ Kaizen พร้อมประสบการณ์จริง
Template Prompt
สมมุติคุณคือ: ผู้เชี่ยวชาญ Kaizen 20 ปี
บริบทองค์กร: [ระบุแผนก/หน่วยงาน]
ผู้ฟัง: [เช่น พนักงานใหม่ / หัวหน้างาน]
สไตล์: [เช่น อบอุ่น เข้าใจง่าย / เป็นทางการ]

คำถาม: [สิ่งที่ต้องการทราบ]
💡 ทำไมได้ผล: การกำหนด Role ชัดเจนทำให้ AI ใช้ประสบการณ์และบริบทที่ถูกต้อง
📐 โครงสร้าง Prompt ที่ดีแบบ Kaizen
ส่วนของ Promptหลักการ Kaizenตัวอย่าง
Role — กำหนดตัวตน AIสวมบทผู้เชี่ยวชาญ"คุณคือ Kaizen expert 20 ปี"
Context — บริบทปัญหาสำรวจสถานการณ์จริง"ในองค์กร 50 คน มีปัญหา..."
Task — สิ่งที่ต้องทำระบุเป้าหมายชัดเจน"วิเคราะห์และเสนอแนะ 3 วิธี"
Format — รูปแบบคำตอบเลิก-ลด-เปลี่ยน output"ตอบเป็นตาราง ไม่เกิน 200 คำ"
Constraint — ข้อจำกัดขอบข่ายข้อเสนอแนะ"ห้ามเสนอซื้อของใหม่ งบ 0 บาท"
🛠️ สร้าง Prompt อัตโนมัติ

✨ เลือกเทคนิค + ใส่ข้อมูล → ได้ Prompt พร้อมใช้

Prompt ที่สร้างขึ้น — คัดลอกไปวางใน AI ได้เลย
